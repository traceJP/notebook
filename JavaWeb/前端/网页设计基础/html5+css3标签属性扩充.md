# 一、HTML标签：

### 1、文字滚动标签：（已弃用）
```html
<marquee behavior="" direction="">
  显示文字
</marquee>
// behavior(滚动方式):scroll(循环滚动);slide(仅滚动一次);alternate(来回交替滚动)
// direction(滚动方向):up;down;left;right
// ...
```

# 二、CSS样式：

### 1、字体间距样式属性：
```css
// 每个字之间间隔多长
letter-spacing: ~px;
```
### 2、字体下划线样式属性：
```css
// 使用该样式的文字会带下划线
text-decoration:
  none无装饰
  blink闪烁
  underline下划线
  line-through贯穿线
  overline上划线
```
3、首行缩进样式属性：
```css
// 当~为n时，首行缩进n个字符
text-indent: ~em;
```
